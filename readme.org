* LearnVulkan

LearnOpenGL but Vulkan implementation

** Cloning

#+BEGIN_SRC bash
git clone https://github.com/K1ngst0m/learnVK

cd learnVK
#+END_SRC

** Building and Running

#+BEGIN_SRC bash
cmake -B build && cmake --build build -- -j$(nproc)

./build/bin/xxx
#+END_SRC
